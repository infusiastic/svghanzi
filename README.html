SVG 漢字

Инструкция по применению

С помощью службы SVG Ханьцзы вы можете получить изображение любого иероглифа в формате SVG.

Для этого достаточно зайти по ссылке вида http://svghanzi.appspot.com/[код иероглифа].

Код иероглифа должен состоять из описательного символа ⿰, ⿱, ⿲, ⿳, ⿴, ⿵, ⿶, ⿷, ⿸, ⿹, ⿺, ⿻ или △ (можно заменить его мнемоникой LR, UD, LCR, UCD, S, SU, SD, SL, SLU, SRU, SLD, OV, TR), за которым следуют компоненты иероглифа.

Описательный символ △ TR — это усовершенствование предложения WERTA для иероглифов из трёх элементов, расположенных треугольником, как 晶 или 轰.

Любой компонент иероглифа может быть выражены в виде +[code point].

Если все компоненты иероглифа одинаковы, можно указать компонент только один раз (Т.е. ⿰木 = ⿰木木, а △木 = △木木木).

Например:

  • http://svghanzi.appspot.com/⿻丨口
  • http://svghanzi.appspot.com/S囗玉

По умолчанию иероглифы выводятся шрифтом 64 пункта. Вы можете задать нужный размер шрифта, добавив к ссылке ?fontsize=[размер в пунктах].

Например:

  • http://svghanzi.appspot.com/⿺辶首?fontsize=64

Вы можете назначить особый шрифт иероглифу, добавив к ссылке?fontfamily=[название шрифта].

Например:

  • http://svghanzi.appspot.com/⿺辶首?fontfamily=serif

В настоящее время служба находится на стадии бета-тестирования. Все вопросы и комментарии, пожалуйста, отправляйте по электропочте автору.

Если вы хотите поддержать развитие сервиса, вы можете сделать пожертвование Магазете с указанием, что передаёте деньги на развитие сервиса SVG Ханьцзы.
